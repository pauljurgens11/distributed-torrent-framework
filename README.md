# Distributed Torrent Framework

Distributed Torrent Simulator is a BitTorrent-inspired peer-to-peer system that models file distribution across a dynamic swarm of nodes. It includes a tracker, configurable peers (seeders and leechers), and a simulation layer for network conditions such as latency, bandwidth limits, and node churn. The system implements core torrent mechanics including piece-based transfer, rarest-first selection, and peer coordination, enabling reproducible experiments and analysis of swarm behavior under varying conditions.

Note: This is a hobby project that's main goal was to discover the intricacies of Torrenting and have some fun!
